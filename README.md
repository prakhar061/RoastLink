# 🔥 RoastLink - An Entertaining Link Shortener App

Welcome to **RoastLink**, my very first full-stack Android project! 🚀 
Instead of boring, traditional short URLs, RoastLink generates entertaining, "suspicious-looking" links (payloads) that spice up the user experience, while seamlessly redirecting users to their destination via a cloud-hosted backend.

## ✨ Features

* **Premium Neon Interface:** Built with a custom Cyan/Neon dark-themed layout.
* **Humorous Payload Slugs:** Automatically attaches funny prefixes like `click-at-your-own-funeral` or `free-virus-installer`.
* **Secure Cloud Synchronization:** Integrated with **Firebase Cloud Firestore** to instantly log and fetch database records.
* **One-Tap Execution:** Leverages the Android Clipboard API to allow one-tap link copies.

---

## 🛠️ System Tech Stack

* **Frontend:** Android SDK, Kotlin, XML Layout Design
* **Database:** Firebase Cloud Firestore (NoSQL Realtime DB)
* **Deployment & Core Routing:** Firebase Hosting, HTML5, Vanilla JavaScript
